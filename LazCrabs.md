# Lazarus crabs

AFK script for all crabs. It's best to start the script at the reset location with the food you would like to eat. 

## Features

* Works everywhere
* Drinks potions
  * Combat
  * Super Combat
  * Strength
  * Super Strength
* Anti logout
* Eats all food
* Logout when out of food
* Special attack
